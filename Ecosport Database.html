Cara Buat Personal Access Token

1

Buka github.com/settings/tokens/new

Langsung klik link ini untuk ke halaman pembuatan token

2

Isi kolom Note

Tulis: ECO SPORT Database

3

Expiration

Pilih No expiration supaya tidak perlu sering ganti

4

Centang repo

Cukup centang 1 checkbox repo di bagian paling atas. Jangan centang yang lain.

repo(Full control of private repositories)
repo:status
repo_deployment
public_repo
repo:invite
security_events
5

Klik Generate token

Scroll ke bawah, klik tombol hijau

6

PENTING: Copy token yang muncul

Token hanya muncul SEKALI! Kalau hilang, harus buat baru. Paste ke kolom Token di halaman utama.

ghp_aBcDeFgHiJkLmNoPqRsTuVwXyZ0123456789

Token ini disimpan di sessionStorage browser kamu saja. Tidak dikirim ke server manapun. Tutup tab = token hilang dari memori.
